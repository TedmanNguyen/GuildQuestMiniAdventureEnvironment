Language Version: Java 17 (JDK 17)

Instructions

1.Execution: Run the program using:  java -jar GuildQuestApp.jar

2.Providing Input: * The program uses a text-based menu system. Enter the number corresponding to the action you wish to take (e.g., type 1 and press Enter to create a campaign).

•When prompted for indices (like selecting a Campaign or a Realm), please enter the integer displayed in the list (starting from 0).

•Note: The program includes input validation to prevent crashes if letters are typed where numbers are expected.
